Teardown Version Manager

Store metadata about specific Teardown builds you own and link scenarios to a required version. The manager won't host or distribute game files — it manages local profile folders and Steam depot references.

  • Define build profiles (Legacy 1.5.x, current, etc.)
  • Reference AppID / DepotID / ManifestID
  • Generate Steam console / depot commands
  • Warn when a scenario's required profile is missing
  • Never asks for Steam credentials in MVP
